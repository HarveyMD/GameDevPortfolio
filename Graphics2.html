Graphics 2
A solo project I built in my second year at university, where I created a 3D rasterization engine from scratch using C++, custom vector/matrix math, and DirectX.

The main feature was terrain rendering using a height map, which allowed for realistic elevation and landscape variation. I also added basic camera controls, so you could move around the 3D space using the keyboard

This project helped me get more comfortable working with 3D space, input handling, and DirectX, while reinforcing a lot of core graphics programming concepts.
