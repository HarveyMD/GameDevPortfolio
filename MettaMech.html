Metta Mech

Metta Mech is a two-player game where mech pilots must work together to fend off an invading alien horde.

It was a final-year university project developed in Unreal Engine 5 by a team of nine members—five programmers and four artists—over 16–20 weeks.

The game was showcased at Derby University's Game Expo 2023 for the public and industry guests.

All programmers worked on separate GitHub Desktop branches to ensure minimal build errors and maintain a smooth development process.

My Contributions to Metta Mech

Project Naming & Team Identity – Named the project and helped define the team’s identity.

Integrity Checks – Implemented boundary-breaking integrity checks to prevent players from escaping the level or exploiting glitches that could break the game.

User Interface (UI) & User Experience (UX) Design – Designed an intuitive UI/UX, including the player menu and controls menu, with input from another programmer and feedback from the artists.

Control Menu – Developed a polished control menu displaying keybindings for both Xbox controllers and keyboard/mouse.

Voice-over Implementation – Created and implemented voice-over lines for key moments, such as round endings and player ammo depletion.

Introductory Panorama – Collaborated with an artist to create an introduction panorama shot for the game’s loading sequence.

Gameplay Features & Effects: Developed item pickups for players, including six types: Ammo, Health, Unlimited Ammo, Unlimited Stamina, Unlimited Health, and Wipe. Implemented an enemy death effect where enemies ragdoll upon defeat and then dissolve over time using a custom shader.
